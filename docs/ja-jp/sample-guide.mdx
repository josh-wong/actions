このサンプルガイドへようこそ！このドキュメントは、様々なドキュメント機能を実演し、適切な書式設定と構造を持つ新しいドキュメント作成のための包括的なテンプレートとして機能します。

## はじめに

このセクションは、プロジェクトを開始するための基本情報を提供します。

### 前提条件

開始する前に、以下を確認してください：

- Markdownの基本的な理解
- リポジトリへのアクセス
- バージョン管理の知識

## 主要機能

このドキュメントシステムは、以下の主要機能をサポートしています：

- **MDX形式** - コンポーネントサポート付きの拡張Markdown
- **自動翻訳** - 自動翻訳機能
- **バージョン管理** - 完全なGit統合
- **プルリクエストワークフロー** - 協調編集プロセス

## 基本的な使用方法

このドキュメントの使用方法は以下の通りです：

1. **作成** - MDX形式でコンテンツを書く
2. **レビュー** - レビュー用のプルリクエストを提出する
3. **マージ** - 承認された変更をマージする
4. **翻訳** - 必要に応じて自動翻訳機能を使用する

## コード例

このようにコードスニペットを含めることができます：

```bash
# 例のコマンド
echo "Hello, World!"
```

```javascript
// JavaScriptの例
function greet(name) {
    return `Hello, ${name}!`;
}
```
